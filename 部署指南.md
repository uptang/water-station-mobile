# 部署到 Vercel 和 Netlify 完整指南

## 🚀 方法一：Vercel 部署（推荐 - 最简单）

### A. 使用 Vercel CLI 部署（最快）

#### 1. 安装 Vercel CLI
```bash
npm install -g vercel
```

#### 2. 登录 Vercel
```bash
vercel login
```
- 会在浏览器中打开登录页面
- 使用 GitHub/GitLab/Bitbucket 账号登录

#### 3. 部署项目
在项目根目录（C:\Users\Administrator\Downloads\test）运行：
```bash
vercel
```

按照提示操作：
- Setup and deploy? **Y**
- Which scope? 选择你的账户
- Link to existing project? **N**
- What's your project's name? **jikedao-water-station**（可自定义）
- In which directory is your code located? **./**
- Want to override the settings? **N**

完成后会得到一个 https://xxxx.vercel.app 链接！

#### 4. 后续更新
修改代码后，只需再次运行：
```bash
vercel --prod
```

---

### B. 使用 Vercel 网站部署（可视化）

#### 1. 准备工作
- 访问 https://vercel.com
- 使用 GitHub/GitLab/Bitbucket 账号注册/登录

#### 2. 上传到 GitHub（推荐）
在项目目录运行：
```bash
# 在 GitHub 创建新仓库后
git remote add origin https://github.com/你的用户名/jikedao-water-station.git
git branch -M main
git push -u origin main
```

#### 3. 在 Vercel 导入项目
1. 点击 "New Project"
2. 选择 "Import Git Repository"
3. 选择你的 GitHub 仓库
4. 配置项目：
   - **Framework Preset**: Vite
   - **Root Directory**: ./
   - **Build Command**: npm run build
   - **Output Directory**: dist
5. 点击 "Deploy"

等待1-2分钟，部署完成！

#### 4. 获取链接
部署成功后会得到：
- Production URL: https://your-project.vercel.app
- 自动 HTTPS
- 全球 CDN 加速

---

### C. 直接上传 dist 文件夹部署

如果不想用 Git：

1. 访问 https://vercel.com
2. 点击 "New Project"
3. 选择 "Deploy from template" 或直接拖拽 `dist` 文件夹到页面
4. 完成部署

---

## 🎨 方法二：Netlify 部署

### A. 使用 Netlify CLI

#### 1. 安装 Netlify CLI
```bash
npm install -g netlify-cli
```

#### 2. 登录
```bash
netlify login
```

#### 3. 部署
在项目根目录运行：
```bash
netlify deploy
```

选择：
- Create & configure a new site? **Y**
- Team: 选择你的团队
- Site name: **jikedao-water-station**（可自定义）
- Publish directory: **dist**

#### 4. 正式发布
```bash
netlify deploy --prod
```

---

### B. 使用 Netlify 网站（拖拽部署 - 最简单）

#### 1. 访问 Netlify
https://www.netlify.com

#### 2. 拖拽部署
1. 注册/登录
2. 点击 "Sites" → "Add new site" → "Deploy manually"
3. **直接拖拽整个 `dist` 文件夹到页面**
4. 等待几秒钟，部署完成！

得到链接：https://random-name.netlify.app

#### 3. 修改域名（可选）
- Site settings → Domain management
- 修改为：https://jikedao-water-station.netlify.app

---

### C. 从 Git 部署

1. 上传代码到 GitHub（同上）
2. Netlify 网站点击 "Import from Git"
3. 选择仓库
4. 配置：
   - **Build command**: npm run build
   - **Publish directory**: dist
5. 点击 "Deploy site"
